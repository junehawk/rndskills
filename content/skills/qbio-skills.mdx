## 소개

qbio-skills는 서울대학교 연구자 환경에 맞춰 제작된 Claude Code 스킬 모음입니다. 한국 공공 연구기관의 행정 시스템(IRIS, SRnD)과 연구 워크플로우(논문 리뷰, 실험노트, 장비 구매)를 AI로 자동화하는 7개의 스킬을 포함합니다.

## 포함된 스킬

### iris-assistant
IRIS(범부처통합연구지원시스템) 과제 보고, 실적 등록, 행정 업무를 자동화합니다. 과제 현황 파악부터 보고서 초안 작성까지 IRIS 관련 반복 작업을 AI 대화로 처리합니다.

### snu-srnd
SRnD(연구행정통합관리시스템) 연구비 지출 결의서 작성, 정산 서류 준비 등 연구비 관련 행정 업무를 자동화합니다. 서울대 연구행정 시스템의 양식과 절차에 최적화되어 있습니다.

### snu-procurement-doc
고가 연구장비 구매 시 필요한 규격서를 자동 생성합니다. 장비 정보를 입력하면 조달청 입찰에 필요한 기술 규격서 초안을 작성해줍니다.

### manuscript-review
투고 전 논문을 13인의 가상 전문가(방법론 전문가, 통계 전문가, 도메인 전문가 등)가 시뮬레이션으로 리뷰합니다. 다양한 관점에서 약점을 미리 파악하고 보완할 수 있습니다.

### paper-interview
논문을 팟캐스트 인터뷰 형식으로 변환합니다. 복잡한 연구 내용을 대화체로 풀어내어 연구 홍보, 랩 세미나 자료, 일반 독자용 요약 등에 활용할 수 있습니다.

### auto-labnote
Claude Code 세션 중 수행한 실험 및 분석 과정을 자동으로 실험노트 형식으로 정리하여 Notion에 게시합니다. 연구자가 별도로 기록하지 않아도 분석 과정이 체계적으로 저장됩니다.

### gmail-autoreply-drafter
미답장 이메일을 탐지하고 맥락에 맞는 답장 초안을 자동으로 작성합니다. 학술 협업 요청, 심사 의뢰, 행정 문의 등 연구자가 받는 다양한 이메일에 대응합니다.

## 설치 방법

```bash
claude skill install https://github.com/pjb7687/qbio-skills
```

## 대상 사용자

서울대 및 국내 공공 연구기관 소속 연구자, 특히 IRIS와 SRnD를 일상적으로 사용하는 PI 및 행정 담당 연구원에게 즉각적인 효용을 제공합니다. Claude Code CLI에 익숙하지 않은 연구자도 각 스킬의 안내에 따라 쉽게 사용할 수 있습니다.
