## 소개

Better IRIS는 IRIS(범부처통합연구지원시스템)의 불편한 사용자 인터페이스를 개선하는 브라우저 유저스크립트입니다. ORCID와의 연동을 통해 논문 메타데이터를 자동으로 입력하고, 반복적인 연구실적 등록 과정을 단순화합니다. Claude Code가 아닌 브라우저에서 직접 실행되는 도구입니다.

## 주요 기능

- **ORCID 연동 자동 입력**: ORCID 계정에 등록된 논문을 IRIS 연구실적 등록 양식에 자동으로 채워넣기
- **논문 메타데이터 자동 완성**: 제목, 저자, 게재지, DOI, 출판연도 등을 수동 입력 없이 자동 입력
- **UI 개선**: IRIS의 불편한 기본 인터페이스를 사용하기 편하게 개선
- **반복 입력 단축**: 자주 사용하는 기관명, 소속 정보 등을 저장하고 재사용

## 설치 방법

1. 브라우저에 [Tampermonkey](https://www.tampermonkey.net/) 확장 프로그램을 설치합니다.
2. Tampermonkey 대시보드에서 새 스크립트 추가를 선택합니다.
3. GitHub 저장소에서 `better-iris.js` 파일의 Raw URL을 복사하여 설치합니다.

```
https://raw.githubusercontent.com/pjb7687/better-iris/main/better-iris.js
```

4. ORCID 계정 연동을 위해 스크립트 설정에서 ORCID ID를 입력합니다.

## 사전 준비

- Tampermonkey 또는 Greasemonkey 브라우저 확장 프로그램
- ORCID 계정 (orcid.org에서 무료 등록)
- IRIS 접속 계정 (소속 기관의 IRIS 계정)

## 대상 사용자

IRIS에 정기적으로 연구실적을 등록해야 하는 PI, 박사후연구원, 대학원생. 특히 논문 수가 많아 실적 등록에 시간이 많이 소요되는 연구자에게 즉각적인 효용을 제공합니다.
