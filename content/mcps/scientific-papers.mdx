## 소개

Scientific Papers MCP는 arXiv, OpenAlex, PubMed Central, Europe PMC, bioRxiv/medRxiv, CORE 등 6개의 주요 학술 데이터베이스를 하나의 MCP 서버로 통합한 도구입니다. Claude와 대화하면서 여러 DB를 동시에 검색하고 논문 전문까지 바로 불러올 수 있습니다.

## 지원 데이터베이스

- **arXiv**: 물리, 수학, CS, 통계, 경제 등 프리프린트 서버
- **OpenAlex**: 2억 건 이상의 학술 논문 메타데이터 오픈 DB
- **PubMed Central**: NIH 생명과학·의학 논문 오픈액세스 아카이브
- **Europe PMC**: 유럽 생명과학 논문 통합 검색
- **bioRxiv / medRxiv**: 생명과학·의학 프리프린트 서버
- **CORE**: 오픈액세스 연구 논문 집성 플랫폼

## 주요 기능

- **통합 논문 검색**: 키워드, 저자, DOI로 6개 DB 동시 검색
- **논문 상세 조회**: 초록, 저자, 인용 수, 게재 정보 확인
- **인용 관계 탐색**: 특정 논문의 인용 논문 및 피인용 논문 목록 조회
- **전문 조회**: 오픈액세스 논문의 본문 전체 불러오기

## 활용 사례

- 연구 주제 선정 전 선행 연구 현황 빠르게 파악
- 특정 방법론의 최신 적용 사례 탐색
- 인용 네트워크를 통한 핵심 참고문헌 자동 발굴
- 논문 리뷰나 연구 제안서 작성 시 관련 문헌 수집

## 설치 방법

```json
{
  "mcpServers": {
    "scientific-papers": {
      "command": "npx",
      "args": ["-y", "@futurelab-studio/latest-science-mcp@latest"]
    }
  }
}
```

Claude Desktop의 `claude_desktop_config.json` 또는 Claude Code의 MCP 설정에 위 내용을 추가하세요.
