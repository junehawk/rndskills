## 소개

kordoc은 HWP, HWPX, PDF 등 한국 공공 및 연구기관에서 널리 사용되는 문서 형식을 마크다운으로 변환하고, 문서 비교·표 추출·양식 인식 등 7가지 도구를 제공하는 MCP 서버입니다.

한국 연구 환경에서 HWP 처리는 필수입니다. 정부 공문, 과제 신청서, 연구 보고서 대부분이 HWP 형식으로 제공되지만 AI 도구는 이를 직접 읽지 못합니다. kordoc은 이 간극을 해소합니다.

## 주요 기능

- **문서 변환 (convert_to_markdown)**: HWP, HWPX, PDF를 마크다운으로 변환
- **문서 비교 (compare_documents)**: 두 문서 간 변경 사항을 구조적으로 비교
- **표 추출 (extract_tables)**: 문서 내 표를 CSV 또는 마크다운 테이블로 추출
- **양식 인식 (recognize_form)**: 공문서 양식의 필드와 내용을 구조화하여 추출
- **텍스트 추출 (extract_text)**: 서식 없이 순수 텍스트만 추출
- **메타데이터 조회 (get_metadata)**: 문서 작성일, 작성자, 버전 등 메타데이터 조회
- **일괄 변환 (batch_convert)**: 폴더 내 다수의 문서를 한 번에 변환

## 설치 방법

```bash
npm install -g kordoc
```

설치 후 MCP 설정에 추가합니다:

```json
{
  "mcpServers": {
    "kordoc": {
      "command": "kordoc",
      "args": ["--mcp"]
    }
  }
}
```

## 활용 사례

- 과제 공고문(HWP)을 Claude에 바로 올려 지원 요건 분석
- 여러 버전의 연구계획서를 비교하여 변경 사항 추적
- 보고서 내 데이터 표를 추출해 분석 작업에 활용
- 정부 공문의 요청 사항을 자동으로 구조화하여 Action Item 정리
