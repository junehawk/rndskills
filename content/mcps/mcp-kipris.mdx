## 소개

mcp_kipris는 한국 특허정보검색서비스(KIPRIS)의 Plus API를 MCP로 래핑하여, Claude와의 대화 중 국내 특허·실용신안·디자인·상표 정보를 직접 검색할 수 있게 해주는 서버입니다.

## 주요 기능

- **특허 검색**: 키워드, 출원인, IPC 분류로 국내 특허 검색
- **특허 상세 조회**: 청구항, 명세서, 도면 정보 조회
- **실용신안 검색**: 실용신안 등록 정보 검색 및 조회
- **디자인 검색**: 디자인 등록 정보 검색
- **상표 검색**: 상표 등록 현황 및 출원인 정보 검색
- **출원인 정보 조회**: 특정 기관/기업의 특허 포트폴리오 파악

## 사전 준비

KIPRIS Plus API 키가 필요합니다. [kipris.or.kr](https://www.kipris.or.kr)에서 회원 가입 후 API 신청을 통해 무료로 발급받을 수 있습니다.

## 설치 방법

```bash
pip install mcp-kipris
```

설치 후 MCP 설정 파일에 API 키를 환경 변수로 전달합니다:

```json
{
  "mcpServers": {
    "kipris": {
      "command": "python",
      "args": ["-m", "mcp_kipris"],
      "env": {
        "KIPRIS_API_KEY": "발급받은_API_키"
      }
    }
  }
}
```

## 활용 사례

- 연구 개발 전 선행 특허 조사로 중복 연구 예방
- 경쟁 기관의 특허 포트폴리오 분석
- 기술 이전 또는 공동 연구 전 지재권 현황 파악
- 연구 성과의 특허 출원 가능성 사전 검토
